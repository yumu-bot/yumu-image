# 用法

绘图程序默认会使用端口 `1611`。

请确保端口不被其他程序占用。

## Windows Bat 运行

```
chcp 65001

set EXPORT_FILE=${你解压素材库的文件路径，比如D:\ExportFileV3\}

yarn start
```

## Linux 运行

你懂的。
