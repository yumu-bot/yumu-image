# 用法

绘图程序默认会使用端口 `1611`。

请确保端口不被其他程序占用。

## Windows 运行

```
chcp 65001

set EXPORT_FILE=${你解压素材库的文件路径，比如 D:\ExportFileV3\}

yarn start
```

## Linux 运行

可以使用 PM2 运行。

你应该能自己解决。
